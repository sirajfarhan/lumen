# Lumen

Six skills that check their own work before handing it to you.

Each one runs on your machine and gets checked before the result reaches you. When a check fails the skill repairs the work and tries again.

## Install

```
/plugin marketplace add sirajfarhan/lumen
/plugin install lumen
```

## Skills

| Skill | Reach for it when |
|---|---|
| writer | The wording carries weight and a flat draft would cost you something |
| expert-research | Being wrong about a decision gets expensive |
| question-sharpener | A discussion keeps circling without reaching what matters |
| concept-visualizer | A picture that looks right but is subtly wrong would mislead people |
| web-builder | You want something real in a browser rather than a mockup |
| skillsmith | A workflow you keep redoing should become permanent |

Unsure which fits? Describe the task and the right one gets picked for you.

## Your data

Submitted content stays out of storage. What gets recorded is how many checks ran, and durable memory persists only where you approved it.

Fuller detail lives in [TRUST.md](./TRUST.md).

## Setup

Skills run without an account and return unchecked output, while checking needs a key — email [hello@siraj.ai](mailto:hello@siraj.ai) to request one — that the plugin asks for when required.
